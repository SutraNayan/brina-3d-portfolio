# BriNa 3D Portfolio

A premium, highly interactive 3D developer and real estate strategy portfolio built with React, Three.js, React Three Fiber, GSAP, and TailwindCSS.

![Demo](public/images/about-abstract.jpg)

## ‚ú® Features
- **Godly 3D Constellation:** A starry WebGL background powered by Three.js featuring Keller Williams brand colors.
- **Dynamic Fluid WebGL Background:** A luxurious, interactive Hero section shader reacting to user scroll and cursor movements.
- **Human-Made Asymmetrical Aesthetics:** Includes hand-drawn SVG string accents, arrows, scribbles, and irregular CSS border masks for an organic, imperfect feel.
- **Physics-Based Hanging Cards:** A beautiful grid of Testimonial cards that dynamically swing into view on scroll using GSAP elastic ease physics.
- **Modern Tech Stack:** Built for performance and fluidity using modern React and WebGL ecosystem tools.

## üöÄ Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation & Running Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/SutraNayan/brina-3d-portfolio.git
   ```
2. Navigate to the `app` directory:
   ```bash
   cd brina-3d-portfolio/app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and navigate to `http://localhost:5173`.
